# API Picker

API Picker 是一个用于浏览和选择 OpenAPI (Swagger) 接口文档并生成精简导出文本的工具。它可以帮助开发者快速查看 API 详情，并筛选出需要的接口信息进行导出。

## 功能特点

- **多源导入**：支持通过 URL 加载或本地上传 JSON 文件的方式导入 OpenAPI 文档。
- **接口浏览**：清晰展示 API 的 HTTP 方法、路径、摘要和标签。
- **搜索与筛选**：支持通过关键字（路径、方法、摘要、标签）快速查找接口。
- **按需选择**：可以勾选特定的接口进行批量操作。
- **自定义导出**：将选中的接口导出为结构化的文本格式，包含请求参数、响应参数及类型信息，方便复制和分享。
